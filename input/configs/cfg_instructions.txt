


pr_fields & issues_fields
-------------------------
    these fields determine what data to ask the API for. They will filter out ANY string that is not
    in the dictionary of available API calls without warning. For PR's, the PR number and merged status
    will be asked for automatically, so they have no provided string.

    For PR's, The available API calls are
        "body": utils.clean_str(cur_pr.body),
        "closed": cur_pr.closed_at.strftime("%D, %I:%M:%S %p"),
        "title": utils.clean_str(cur_pr.title),
        "userlogin": lambda cur_pr: cur_pr.user.login,
        "username": lambda cur_pr: cur_pr.user.name,
